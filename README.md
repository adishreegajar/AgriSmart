# 🌾 AgriSmart – AI Powered Smart Agriculture Platform

AgriSmart is a full-stack web application designed to help farmers make data-driven agricultural decisions using real-time weather data, crop market insights, and smart recommendations.

---

## 🚀 Features

- 🌦️ Live Weather Updates (Temperature, Humidity, Rainfall, Wind)
- 🌾 Crop Marketplace (Farmer-to-Buyer direct interaction)
- 📊 Smart Dashboard with analytics
- 🤖 Crop Recommendation System (Rule-based AI)
- 🏛️ Government Schemes Information
- 🔐 User Authentication (Login/Signup)

---

## 🧠 AI Component

The platform includes a rule-based recommendation engine that suggests crops based on:

- Weather conditions  
- Market trends  

Example:

If rainfall is high → Recommend Rice  
If dry weather + high demand → Recommend Soybean
